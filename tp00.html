¿Que es HTML y en que consiste?

HTML es el componente más basico de toda la WEB. Este mismo consiste en dar y definir el significado y la estructura a un contenido WEB

¿Que es una etiqueta? ¿ Funciona tanto en mayuscula como en minuscula?

Una etiqueta nos sirve para poder crear cualquier elemento en HTML. Si,funciona tanto en mayuscula como en minuscula.

¿A que llamamos elemento HTML? ¿Cuál es su anatomia? Explicar sus partes

Llamamos elemento en HTML a la raiz del documento HTML. Su anatomia consiste en: La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (<>) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento —en este caso, dónde es el comienzo del párrafo—. La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso dónde termina el párrafo—. El contenido: este es el contenido del elemento, que en este caso es sólo texto. El elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

¿Que es un atributo? Dar 2 ejemplos

Los atributos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios. Ejemplos: "charset="UTF-8"" y "html lang="en"""

¿Que hace la etiqueta P,i,strong,em,sup,sub,u,br?

Las etiquetas "p" sirven para crear los párrafos y dentro de ellas también se indican atributos específicos, como la alineación del texto, ya sea izquierda, derecha, centrada o justificada

La etiqueta "i" muestra el texto marcado con un estilo en cursiva o italica. Ejemplo Estoy en taller de WEB

La etiqueta "strong" marcar con especial énfasis las partes más importantes de un texto. Taller web

La etiqueta "sup" define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice. Taller web

La etiqueta "u" poner un texto subrayado de cara tanto visualmente para el lector como internamente para los buscadores. Ejemplo River Plate

La etiqueta "br" produce un salto de línea en el texto. Hoy es
jueves

La etiqueta "sub" efine un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. Ejemplo Taller de web

La etiqueta "em" marcar con énfasis las partes importantes de un texto. Taller web
